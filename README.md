# Aklilu Legesse Portfolio

Premium cinematography portfolio website for Aklilu Legesse.

## Files

- `index.html` - main portfolio page
- `styles.css` - visual design and responsive layout
- `script.js` - motion, cursor interactions, and video behavior
- `preview-server.js` - simple local preview server

## Local Preview

Open the local preview at `http://127.0.0.1:4173` when the preview server is running.

## Deploy

This project can be deployed as a static site on GitHub Pages, Vercel, or Netlify.
